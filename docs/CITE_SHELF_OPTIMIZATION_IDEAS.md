# 文献篮功能优化建议（分阶段）

更新时间：2026-03-07

## 目标

让文献篮从“临时收藏区”升级为“可复用的研究工作台”。

## P1（1-2 周，低风险高价值）

1. 批量操作
- 批量移除、批量复制（GB/T、BibTeX）、批量加入“导出队列”。

2. 快速检索与排序
- 文献篮内搜索（标题/作者/DOI）。
- 排序：最近加入、被引次数、年份、IF/CORE/CCF。

3. 去重与合并提示
- 同 DOI 自动去重。
- 同标题近似去重（提示“可能重复”，可手动合并）。

4. 会话来源追踪
- 每条文献显示“来自哪个会话/哪次回答/哪个引用点”。

5. 条目一句话概括（点击即见）
- 点击文献篮条目时，在详情区显示“1 句话概括”。
- 展示概括来源标记：`fulltext` / `abstract` / `metadata`，避免用户误判可信度。
- 无高质量摘要时，回退为保守描述（标题 + 研究对象/方法关键词），不胡编细节。

## P2（2-4 周，中等改造）

1. 标签与分组
- 自定义标签（`baseline`/`idea`/`related-work`）。
- 分组视图（按任务、按会话、按标签）。

2. 文献篮快照
- 一键保存当前篮子为“快照版本”。
- 支持比较两个快照的新增/移除差异。

3. 质量提示
- 对低质量条目打标（缺标题、缺 DOI、元数据冲突）。
- 提供“修复建议”入口（重新匹配、手工编辑）。

4. 导出中心
- 导出为 `.bib`、`.ris`、`.csv`。
- 支持“仅导出已勾选”。

## P3（4+ 周，产品化能力）

1. 项目级文献篮
- 从“按会话隔离”扩展到“按项目隔离 + 会话关联”。

2. 关联笔记
- 每条文献可附加“阅读笔记/结论/待验证点”。
- 支持从回答中一键追加笔记。

3. 协作与分享
- 生成只读分享链接或导出包。
- 团队成员可合并建议文献篮。

## 建议优先级（执行顺序）

1. `批量操作 + 搜索排序 + 去重 + 点击显示一句话概括`（立即提升可用性）
2. `标签分组 + 快照`（提升研究流程稳定性）
3. `项目级管理 + 协作`（中长期产品化）

## 验收指标（建议）

1. 文献篮操作时长中位数下降（加入后找到并复制条目）。
2. 重复条目比例下降。
3. 导出成功率与可用率提升（导出后可直接在 Zotero/EndNote 使用）。
4. 文献篮详情点击后的有效信息到达时间下降（用户在 3 秒内看到可读概括）。
5. 一句话概括可用率提升（有概括且来源可信的条目占比持续上升）。
